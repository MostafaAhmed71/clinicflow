ClinicFlow منصة سحابية عربية بالكامل (RTL) مصمَّمة لعيادة مصر اليومية. تدير ملفات المرضى، مكتب السكرتارية، غرفة الانتظار الحية، شاشة الكشف والروشتة، طلبات التحاليل والأشعة، تعليمات المريض، الفواتير والخزنة، والتقارير — مع فصل واضح بين دور الطبيب والسكرتارية، ودعم الكمبيوتر والموبايل (PWA).

ماذا يقدّم النظام بالتفصيل؟

١) مكتب الحجوزات والسكرتارية

  • بحث مريض بالاسم/الهاتف وإضافة سريعة
  • حجز كشف جديد أو إعادة كشف بأسعار منفصلة
  • غرفة انتظار: انتظار / مع الطبيب / منتهي / غياب
  • متابعة غير المدفوع بعد انتهاء الكشف
  • زر واتساب برسالة تذكير جاهزة

٢) شاشة الكشف للطبيب

  • وضع تركيز يخفي القوائم ويظهر المريض والطابور
  • تاريخ مرضي + حساسية + علامات حيوية
  • آخر زيارات بجانب الكشف
  • روشتة مع بحث أدوية مصرية وجرعات جاهزة
  • قوالب تعليمات حسب تخصص العيادة

٣) الطباعة والهوية

  • روشتة بشعار وختم وتوقيع
  • طلب تحاليل وطلب أشعة منظم للمعامل
  • ورقة تعليمات خروج للمريض
  • دعم A4 أو طابعة حرارية من الإعدادات

٤) الإدارة والتقارير

  • لوحة: مرضى اليوم، انتظار، غياب، إيراد، متابعات
  • تقارير مرضى / إيراد / طبيب / أمراض
  • فواتير وخصومات وخزنة يومية
  • إقفال يوم وتقرير أسبوعي قابل للطباعة

لماذا يختلف عن «برنامج مواعيد» عادي؟

  • يربط الحجز بالكشف بالطباعة بالتحصيل في حلقة واحدة
  • يتكيّف مع تخصص العيادة (حقول وقوالب وفحوصات)
  • يفصل صلاحيات الطبيب عن السكرتارية بوضوح
  • مصمم لسيناريو مصر: كاش، حراري، واتساب، Excel
  • Realtime يحدّث الانتظار بين الأجهزة فورًا
  • يعمل كتطبيق على الموبايل من الشاشة الرئيسية

ملفات المرضى والكارت الطبي

كل مريض له رقم ملف تلقائي داخل العيادة — مع استيراد جماعي من Excel لترحيل العيادة القديمة.

بيانات أساسية

  • الاسم، الهاتف، الرقم القومي، تاريخ الميلاد، النوع
  • المهنة، العنوان، الحالة الاجتماعية، فصيلة الدم
  • التأمين، جهة اتصال الطوارئ
  • بحث سريع بالاسم أو الهاتف أثناء المكالمة

التاريخ الطبي

  • أمراض مزمنة، عمليات، أمراض وراثية
  • حساسية أدوية (تنبيه أثناء الروشتة)
  • تدخين / كحول / حمل / تطعيمات
  • علامات حيوية: ضغط، سكر، وزن، حرارة، نبض، أكسجين

سجل الزيارات

  • خط زمني لكل كشف: شكوى، فحص، تشخيص، خطة
  • الروشتات والطلبات المرتبطة بالزيارة
  • مرفقات: صور، PDF، نتائج تحاليل/أشعة
  • موعد المتابعة الظاهر في صندوق المتابعات

ترحيل العيادة

  • استيراد مرضى من ملف Excel
  • هدف الترحيل: تشغيل العيادة خلال ساعة
  • بيانات كل عيادة معزولة عن باقي العيادات

الحجوزات وغرفة الانتظار

الحجز من المكتب

  • اختيار مريض موجود أو إنشاء سريع (اسم + هاتف)
  • تحديد الطبيب والمدة ووقت الموعد
  • نوع الزيارة: كشف جديد أو إعادة كشف
  • رسوم تلقائية حسب سعر العيادة لكل نوع
  • قائمة حجوزات اليوم مع الحالات

غرفة الانتظار الحية

  • تحديث فوري بين جهاز السكرتارية والطبيب (Realtime)
  • حالات ملونة: انتظار · مع الطبيب · منتهي · غياب
  • بانر المريض التالي + زمن الانتظار
  • فلاتر سريعة حسب الحالة
  • إجراءات: استدعاء، تأكيد دفع، واتساب، المزيد

شاشة الكشف — قلب النظام

مصمَّمة لتقليل الثواني بين مريض ومريض، مع بقاء السياق الطبي أمامك.

وضع التركيز (Focus)

  • إخفاء الشريط الجانبي أثناء الكشف
  • المريض الحالي + الطابور في نفس الشاشة
  • زر الانتقال للمريض التالي بعد الحفظ
  • ملخص سريع: حساسية، أمراض مزمنة، آخر تشخيص

محتوى الزيارة

  • الشكوى الحالية وتاريخ المرض الحالي
  • الفحص السريري والتشخيص وخطة العلاج
  • ملاحظات + حقول إضافية حسب التخصص
  • موعد المتابعة

الروشتة

  • بحث أدوية مصرية (قاعدة محلية / استيراد)
  • جرعة ومدة من قوائم جاهزة + مخصص
  • «عند اللزوم» وخيارات شائعة للعيادة المصرية
  • تنبيه أحمر عند تعارض دواء × حساسية مسجّلة
  • مفضلات الطبيب وقوالب روشتة
  • طباعة فورية بعد الحفظ

الطلبات والتعليمات

  • كتالوج تحاليل حسب تخصص العيادة
  • كتالوج أشعة/تصوير حسب التخصص
  • إمكانية فتح تخصص آخر عند الحاجة
  • قوالب تعليمات جاهزة لكل تخصص + عامة
  • ورقة خروج المريض للطباعة

قائمة مميزات الكشف باختصار

  • عرض التخصص الحالي في رأس شاشة الكشف
  • اقتراح تحاليل شائعة للتخصص بضغطة
  • طلب جراحة/إجراء كنص حر عند الحاجة
  • حفظ الزيارة مع ربط الموعد وحالة الدفع
  • طباعة روشتة / تعليمات / تحاليل / أشعة من نفس الشاشة
  • دعم طباعة A4 أو حراري حسب إعداد العيادة

التخصصات الطبية

تختار تخصص العيادة مرة واحدة من الإعدادات — الواجهة تتكيّف: حقول كشف، تحاليل مقترحة، قوالب تعليمات، وكتالوج فحوصات مناسب.

أمثلة على التكيّف

  • أطفال: نمو/تطعيمات + تعليمات حرارة وجرعات

  • جلدية: موضع الإصابة + تعليمات مراهم
  • نساء: آخر دورة / عمر الحمل + تعليمات حمل
